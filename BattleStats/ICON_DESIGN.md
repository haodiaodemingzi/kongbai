# 应用图标设计方案

## 🎨 设计理念

### 主题：战斗统计
- 核心元素：剑、盾、统计图表
- 色彩：紫色渐变（与登录页一致）
- 风格：现代、简洁、专业

## 📐 图标规格

### 所需图标文件

1. **icon.png** - 1024x1024
   - 应用主图标
   - 用于应用商店展示

2. **adaptive-icon.png** - 1024x1024
   - Android 自适应图标前景
   - 透明背景

3. **splash-icon.png** - 1284x2778
   - 启动屏幕图标
   - 可以是简化版的 logo

4. **favicon.png** - 48x48
   - Web 版图标

## 🎯 设计方案

### 方案一：剑盾组合（推荐）
```
┌─────────────┐
│   ⚔️ 🛡️    │  剑与盾交叉
│             │
│   📊        │  下方统计图标
│             │
│ Battle Stats│  文字标识
└─────────────┘
```

**特点**:
- 直观表达战斗主题
- 图标清晰易识别
- 紫色渐变背景

### 方案二：统计图表（简约）
```
┌─────────────┐
│             │
│   📈 ⚔️     │  上升趋势 + 剑
│             │
│     BS      │  缩写
└─────────────┘
```

**特点**:
- 强调数据统计
- 简洁现代
- 适合小尺寸显示

### 方案三：圆形徽章
```
    ╭─────╮
   │  ⚔️  │    中心剑图标
   │ ─── │    装饰线条
   │  📊  │    底部图表
    ╰─────╯
```

**特点**:
- 圆形设计更柔和
- 适合 Android 自适应图标
- 紫色渐变边框
